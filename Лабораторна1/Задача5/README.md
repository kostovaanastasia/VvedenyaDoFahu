 Задача 5
Напишіть програму, яка приймає оцінку студента (ціле число від 0 до 100)  і виводить оцінку за шкалою: «відмінно», «добре», «задовільно» або  «незадовільно». 

Опис роботи: Програма приймає від користувача оцінку студента — ціле число від 0 до 100. На основі введеного значення програма визначає рівень знань за чотирибальною шкалою: «відмінно», «добре», «задовільно» або «незадовільно». Виведення відбувається одразу після перевірки умов.

Алгоритм (блок-схема):
Вхід → Ввести оцінку a → Перевірити (a < 60) → Вивести «незадовільно»
Інакше → Перевірити (60–73) → Вивести «задовільно»
Інакше → Перевірити (74–89) → Вивести «добре»
Інакше → Перевірити (90–100) → Вивести «відмінно» → Вихід

Синтаксис використаних операторів C++:
cin >>, cout <<, int, if, else if, логічні оператори <, >, &&

Пояснення коду: Програма оголошує змінну a типу int, у яку записується введена користувачем оцінка. Через послідовність умовних операторів if та else if оцінка перевіряється на належність до певного діапазону.
• Якщо оцінка менша за 60 — виводиться «Незадовільно».
• Якщо значення від 60 до 73 — виводиться «Задовільно».
• Якщо від 74 до 89 — «Добре».
• Якщо від 90 до 100 — «Відмінно».
Після виведення програма завершується.

Інструкція використання:
Запустити програму.
Ввести число в діапазоні 0–100.
Отримати відповідну оцінку рівня знань за текстовою шкалою.

Приклад введення: Твоя оцінка: 57
Очікуваний результат: Незадовільно
