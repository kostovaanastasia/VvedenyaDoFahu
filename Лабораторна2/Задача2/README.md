Задача 2 

Перевірка віку для доступу до сайту/послуги. Розробити програму, яка перевіряє, чи дозволено користувачу доступ на сайт або до послуги за віком: − Вік < 13 → доступ заборонено 
− Вік 13–17 → доступ з батьківським контролем 
− Вік ≥ 18 → повний доступ 

Алгоритм (блок-схема):
Вхід → Ввести вік → Перевірити умови:
age < 13 → Вивести «Доступ заборонено» → Вихід
13 ≤ age < 18 → Вивести «Доступ з батьківським контролем» → Вихід
age ≥ 18 → Вивести «Повний доступ» → Вихід

Синтаксис використаних операторів мови C++:
cin >>, cout <<, if, else if, else, int, логічні оператори <, >=, &&

Пояснення коду:
Оголошується змінна age типу int для зберігання віку користувача.
Програма запитує у користувача вік через cin.
Використовуються умовні оператори для перевірки віку:
Якщо вік менше 13, доступ заборонено.
Якщо вік від 13 до 17 включно, доступ можливий з батьківським контролем.
Якщо вік 18 і більше, надається повний доступ.
Програма завершує роботу після виведення повідомлення про рівень доступу.

Інструкція використання:
Запустити програму.
Ввести свій вік.
Отримати повідомлення про рівень доступу до сайту або послуги.

Приклад введення: Ваш вік: 17
Очікуваний результат: Доступ з батьківським контролем
