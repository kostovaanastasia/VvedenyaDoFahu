Напишіть програму, яка запитує у користувача ціле число і виводить на  екран залишок при діленні введеного числа на 7. Наступним кроком програма  повинна запросити у користувача два цілих числа і вивести повідомлення чи  ділиться перше число на друге без залишку, при наявності залишку вказати його  в повідомленні. Після чого перевірити дані числа на парність та вивести  інформаційне повідомлення про результат перевірки (наприклад, перше число 8  – парне, друге число 17 – непарне). 
Опис роботи: Програма спочатку запитує у користувача ціле число і виводить залишок при діленні цього числа на 7. Потім вона просить ввести два цілі числа і перевіряє, чи ділиться перше число на друге без залишку, або виводить залишок при діленні. Після цього програма визначає парність кожного з двох введених чисел і виводить відповідне повідомлення.
Алгоритм (блок-схема): Вхід -> Ввести a -> Обчислити a % 7 -> Вивести результат -> Ввести num1, num2 -> Перевірити ділення без залишку -> Вивести результат -> Перевірити парність num1 -> Вивести -> Перевірити парність num2 -> Вивести -> Вихід
Cинтаксис використаних операторів мови С++: cin >>, cout <<, %, if, else
Пояснення коду: Оголошується змінна a типу int. Через cout програма підказує користувачу ввести число. Зчитування числа через cin >> a. Обчислення залишку при діленні на 7 через a % 7 і збереження у змінну b типу double для зручного відображення.Виведення залишку через cout Оголошуються дві змінні num1 і num2 типу int. Підказка користувачу ввести два числа та зчитування через cin >> num1 >> num2. Перевірка ділення першого числа на друге без залишку через if (num1 % num2 == 0). Виводиться відповідне повідомлення, або обчислюється залишок і виводиться через cout.Перевірка парності першого числа: якщо num1 % 2 == 0, число парне, інакше непарне. Аналогічно перевірка для другого числа. Вивід повідомлень про парність обох чисел.
Інструкція використання:Запустити програму, ввести ціле число для перевірки залишку при діленні на 7, потім ввести два числа для перевірки ділення без залишку та парності, отримати відповідні повідомлення.
Приклад введення: Введіть ціле число: 67 Введіть два цілих числа: 43 28
Очікувані результати:
a % 7 = 4
Ділиться з залишком: 15
Перше число 43 непарне
