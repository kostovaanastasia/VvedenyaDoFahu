# ВведенняДоФаху
Репозиторій для виконання лабораторних робіт з  вивчення мови програмування С++
Дисципліна призначена для вивчення основних основ аспектів сучасних комп’ютерних технологій
Як запускати роботи: Написати код в VSCode, зберегти файл, запустити програму, завантажити через Git
Вимоги до коду: читабельність, відступи, логічність, код повинен скомпілюватись без помилок
Оцінювання за 100-бальною шкалою: 
90-100 - Відміно 
82-89 - Добре 
74-81 - Добре 
63-73 - Задовільно 
35-59 - Незадовільно з можливістю повтороного складання 
0-34 - Незадовільно з обов'язковим повторним вивченням дисципліни
