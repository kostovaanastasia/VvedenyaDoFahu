Задача 3
Банкомат – перевірка суми для зняття готівки. Програма перевіряє, чи  можна зняти введену суму з банківського рахунку: 
− Сума ≤ 0 → помилка 
− Сума > баланс → недостатньо коштів 
− Інакше → операція успішна

Алгоритм (блок-схема):
Вхід → Ввести суму → Перевірити умови:
suma ≤ 0 → Вивести «Помилка» → Вихід
suma > balance → Вивести «Недостатньо коштів» → Вихід
Інакше → Вивести «Операція успішна» → Вихід

Синтаксис використаних операторів мови C++:
cin >>, cout <<, if, else if, else, int, const, логічні оператори <=, >.

Пояснення коду:
Задається константа balance для зберігання доступного балансу.
Оголошується змінна suma типу int для введеної користувачем суми.
Виконується перевірка:
Якщо введена сума менше або дорівнює нулю, виводиться «Помилка».
Якщо сума більша за баланс, виводиться «Недостатньо коштів».
Інакше операція успішна, і виводиться відповідне повідомлення.

Інструкція використання:
Запустити програму.
Ввести суму для зняття.
Отримати повідомлення про результат операції: помилка, недостатньо коштів або успішне зняття.

Приклад введення: Сума: 0
Очікуваний результат: Помилка

Приклад введення: Сума: 789
Очікуваний результат: Операція успішна
