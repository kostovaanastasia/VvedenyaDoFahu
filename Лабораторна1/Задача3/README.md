Напишіть програму, яка приймає ціле число Х та виводить на екран число до виконання інкрементації, після виконання префіксної та постфіксної  інкрементації. Повторити рішення для декрементації, під час обчислень в якості  значення використати введене значення Х (без врахування​​ змін інкрементації). Далі, використати введене значення Х для обчислення Y, наприклад: Y = 10 + X.  Виконати обчислення Y = 10 + X з префіксною та постфіксною інкрементацією X (наприклад: Y = 10 + ++X, Y = 10 + X++). Після кожної операції обчислення  виводити значення X та Y.
Опис роботи: Програма спочатку приймає ціле число X і демонструє роботу префіксної та постфіксної інкрементації і декрементації, використовуючи введене значення X. Далі програма обчислює значення Y = 10 + X та повторює обчислення з префіксною і постфіксною інкрементацією X, після кожної операції виводячи поточні значення X та Y.
Алгоритм (блок-схема):
Вхід -> Ввести X -> Зберегти X в x -> Префіксна інкрементація -> Вивести -> Постфіксна інкрементація -> Вивести -> Префіксна декрементація -> Вивести -> Постфіксна декрементація -> Вивести -> Обчислити Y = 10 + X -> Вивести -> Y = 10 + ++X -> Вивести -> Y = 10 + X++ -> Вивести -> Вихід
Cинтаксис використаних операторів мови С++: cin >>, cout <<, int, ++, --, =, +
Пояснення коду: Оголошується змінна X типу int. Через cout підказуємо користувачу ввести число.Зчитування числа через cin >> X. Зберігаємо початкове значення X у змінну x, щоб використовувати його повторно без впливу інкрементації/декрементації. Префіксна інкрементація: ++X — спочатку збільшує X, потім присвоює результат змінній X1. Виводимо X1. Відновлюємо X = x, робимо постфіксну інкрементацію X++, присвоюємо X2, виводимо. Аналогічно виконуємо префіксну та постфіксну декрементацію --X та X--. Виконуємо обчислення Y = 10 + X для початкового значення X.Повторюємо обчислення з префіксною інкрементацією Y = 10 + ++X і постфіксною Y = 10 + X++. Після кожної операції виводимо поточні значення X і Y.
Інструкція використання: Запустити програму, ввести ціле число X. Отримати результати префіксної та постфіксної інкрементації та декрементації, а також обчислення Y = 10 + X з різними варіантами інкрементації X.
Приклад введення: X: 3

Постфіксна декрементація: 3
