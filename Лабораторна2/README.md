Задача 1

Погодний порадник. Програма радить, що взяти з собою на вулицю за температурою: − Температура ≤ 0 – «Беремо теплий одяг та рукавички» − 1–15 – «Візьміть легку куртку» − 16–25 – «Можна одягати футболку» − 25 – «Легкий одяг, сонцезахисні окуляри»

Опис роботи: Програма приймає від користувача значення температури та радить, що варто взяти з собою на вулицю залежно від температурного діапазону

Алгоритм (блок-схема):
Вхід → Ввести температуру → Перевірити умови:
C ≤ 0 → Вивести «Беремо теплий одяг та рукавички» → Вихід
0 < C ≤ 15 → Вивести «Візьміть легку куртку» → Вихід
15 < C ≤ 25 → Вивести «Можна одягати футболку» → Вихід
C > 25 → Вивести «Легкий одяг, сонцезахисні окуляри» → Вихід

Синтаксис використаних операторів мови C++:
cin >>, cout <<, if, else if, else, int, логічні оператори <=, >, &&

Пояснення коду:
Спочатку оголошується змінна C типу int для зберігання температури, яку вводить користувач. Програма виводить повідомлення "Температура: " і зчитує число через cin >> C. Далі застосовуються умовні оператори if та else if: якщо C менше або дорівнює 0, виводиться "Беремо теплий одяг та рукавички"; якщо від 1 до 15 включно, "Візьміть легку куртку"; від 16 до 25 включно, "Можна одягати футболку"; якщо більше 25, "Легкий одяг, сонцезахисні окуляри". Програма перевіряє умови зверху вниз і виконує перший блок коду, умова якого істинна. Після виведення поради програма завершує роботу з return 0;. Таким чином, користувач вводить температуру, а програма автоматично радить, який одяг брати на вулицю.

Інструкція використання:
Запустити програму.
Ввести температуру у градусах Цельсія.
Отримати рекомендацію щодо одягу.

Приклад введення: Температура: 19
Очікуваний результат: Можна одягати футболку


Задача 2 

Перевірка віку для доступу до сайту/послуги. Розробити програму, яка перевіряє, чи дозволено користувачу доступ на сайт або до послуги за віком: − Вік < 13 → доступ заборонено 
− Вік 13–17 → доступ з батьківським контролем 
− Вік ≥ 18 → повний доступ 

Алгоритм (блок-схема):
Вхід → Ввести вік → Перевірити умови:
age < 13 → Вивести «Доступ заборонено» → Вихід
13 ≤ age < 18 → Вивести «Доступ з батьківським контролем» → Вихід
age ≥ 18 → Вивести «Повний доступ» → Вихід

Синтаксис використаних операторів мови C++:
cin >>, cout <<, if, else if, else, int, логічні оператори <, >=, &&

Пояснення коду:
Спочатку оголошується змінна age типу int для зберігання введеного віку. Далі програма виводить повідомлення "Ваш вік: " і зчитує число через cin >> age. Потім використовуються умовні оператори if та else if для перевірки віку: якщо age менше 13, виводиться "Доступ заборонено"; якщо від 14 до 17 включно, "Доступ з батьківським контролем"; якщо 18 і більше, "Повний доступ". Програма перевіряє умови зверху вниз і виконує перший блок коду, умова якого істинна, ігноруючи інші. Після виведення повідомлення програма завершує роботу з return 0;. Таким чином, користувач вводить свій вік, а програма автоматично повідомляє, чи доступ дозволено повністю, частково, чи заборонено.

Інструкція використання:
Запустити програму.
Ввести свій вік.
Отримати повідомлення про рівень доступу до сайту або послуги.

Приклад введення: Ваш вік: 17
Очікуваний результат: Доступ з батьківським контролем

                            Задача 3
Банкомат – перевірка суми для зняття готівки. Програма перевіряє, чи  можна зняти введену суму з банківського рахунку: 
− Сума ≤ 0 → помилка 
− Сума > баланс → недостатньо коштів 
− Інакше → операція успішна

Алгоритм (блок-схема):
Вхід → Ввести суму → Перевірити умови:
suma ≤ 0 → Вивести «Помилка» → Вихід
suma > balance → Вивести «Недостатньо коштів» → Вихід
Інакше → Вивести «Операція успішна» → Вихід

Синтаксис використаних операторів мови C++:
cin >>, cout <<, if, else if, else, int, const, логічні оператори <=, >.

Пояснення коду:
Задається константа balance для зберігання доступного балансу.
Оголошується змінна suma типу int для введеної користувачем суми.
Виконується перевірка:
Якщо введена сума менше або дорівнює нулю, виводиться «Помилка».
Якщо сума більша за баланс, виводиться «Недостатньо коштів».
Інакше операція успішна, і виводиться відповідне повідомлення.

Інструкція використання:
Запустити програму.
Ввести суму для зняття.
Отримати повідомлення про результат операції: помилка, недостатньо коштів або успішне зняття.

Приклад введення: Сума: 0
Очікуваний результат: Помилка

Приклад введення: Сума: 789
Очікуваний результат: Операція успішна

                              Задача 4 
 Визначення оцінки студента за балами Програма перетворює бали на  оцінку: 
− 90–100 → «Відмінно» 
− 75–89 → «Добре» 
− 60–74 → «Задовільно» 
− <60 → «Незадовільно» 

Алгоритм (блок-схема):
Вхід → Ввести бали → Перевірити умови:
a < 60 → Вивести «Незадовільно» → Вихід
60 ≤ a ≤ 74 → Вивести «Задовільно» → Вихід
75 ≤ a ≤ 89 → Вивести «Добре» → Вихід
90 ≤ a ≤ 100 → Вивести «Відмінно» → Вихід

Синтаксис використаних операторів мови C++:
cin >>, cout <<, if, else if, else, int, логічні оператори <, <=, >, &&

Пояснення коду:
Спочатку оголошується змінна a типу int для зберігання балів, які вводить користувач. Далі програма виводить повідомлення "Твоя оцінка: " і через cin >> a зчитує введене число. Після цього використовується послідовність умовних операторів if та else if для визначення оцінки: якщо бали менші за 60, виводиться "Незадовільно"; якщо від 60 до 74 включно, "Задовільно"; від 75 до 89 – "Добре"; від 90 до 100 – "Відмінно". Логічний оператор && використовується для поєднання двох умов у межах одного діапазону. Програма перевіряє умови зверху вниз і виконує перший блок коду, умова якого істинна, ігноруючи всі наступні. Після виведення відповідної оцінки програма завершує роботу з return 0;. Таким чином, користувач вводить свої бали, а програма автоматично виводить відповідну оцінку за шкалою «Незадовільно», «Задовільно», «Добре» або «Відмінно».

Інструкція використання:
Запустити програму.
Ввести кількість балів студента.
Отримати оцінку за шкалою.

Приклад введення: Твоя оцінка: 95
Очікуваний результат: Відмінно
